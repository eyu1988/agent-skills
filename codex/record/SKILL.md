# record

## 路径常量

- 日捕获目录：`{{CAPTURE_DIR}}`
- 模板文件：`{{CAPTURE_DIR}}/_模板.md`

## 触发方式

- `记录一下：[内容]`
- `记录：[内容]`
- `/record [内容]`
- 对话中直接描述（提取核心内容）

## 执行步骤

### 第一步：确定今日文件路径

```bash
date +%Y-%m-%d
```
目标文件：`{{CAPTURE_DIR}}/YYYY-MM-DD.md`

### 第二步：文件不存在则创建

```bash
[ ! -f "{{CAPTURE_DIR}}/YYYY-MM-DD.md" ] && cp "{{CAPTURE_DIR}}/_模板.md" "{{CAPTURE_DIR}}/YYYY-MM-DD.md"
```

然后将文件中的 `YYYY-MM-DD` 占位符替换为今天日期。

### 第三步：提取用户输入

从触发语句中提取要记录的核心内容。

### 第四步：判断类型标签

根据内容自动打上类型标签：
- `#技术` — 工具用法、API、实现细节
- `#概念` — 定义、原理、理论
- `#实践` — 做了什么、踩了什么坑
- `#决策` — 选了哪个方案及原因
- `#想法` — 灵感、待验证的假设
- `#问题` — 还没解答的疑问

### 第五步：追加写入

在今日文件"📥 今日捕获"区域最后一个 `-` 之后追加一行：

```
- [标签] [内容] | [为什么当时在意，一句话]
```

若用户没有说明"为什么在意"，根据上下文自动补一句简短的原因。

### 第六步：确认

输出一行确认：`已记录 → [今日文件名]`
