# 输出模板

## 快速判断

用于用户只需要简短判断时。

```markdown
**结论**
...

**理由**
- ...
- ...

**主要风险**
...
```

## 严谨决策审查

用于重要的商业、投资、职业、产品、技术或组织决策。

```markdown
**决策**
...

**结论**
...

**使用的模型**
- ...

**反向审查**
- ...

**证据**
- 事实：
- 假设：
- 未知：

**风险**
- ...

**什么会改变我的判断**
- ...

**下一步行动**
...
```

## 反方审查

用于用户问“我漏了什么”“这个判断哪里可能错”“帮我挑战一下”时。

```markdown
**最大风险**
...

**失败路径**
- ...

**可能偏误**
- ...

**缺失证据**
- ...

**更稳妥的版本**
...
```

## 模型地图

用于用户询问“这个问题适合用哪些模型分析”时。

```markdown
**最相关模型**
- 模型：为什么它对这里重要。

**次相关模型**
- 模型：为什么它只是辅助。

**不要过度使用**
- 模型：为什么它可能误导当前判断。
```
